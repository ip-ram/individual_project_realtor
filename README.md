# individual_project_realtor

Консольное Python-приложение для работы с базой квартир риэлтерского агентства.

## Возможности

- Загрузка квартир из aprtment.json с валидацией данных

- Сортировка методом Шелла

## Просмотр:

- Всех квартир (комнаты ↓, цена ↑)

- Квартир с заданным числом комнат (этаж ↑, этажность ↑, цена ↓)

- Квартир в диапазоне цен (цена ↓, общая площадь ↑)

## Файлы

- index.py — основной файл программы

- json_methods.py — чтение и проверка JSON

- sort_methods.py — реализации сортировок (Shell, Bubble)

Запуск
```bash
python index.py
```
